# UsageGuard Next Steps (Public)

This is the public high-level roadmap.

## Next priorities
1. Provider schema parity hardening with real production payloads.
2. Retry/backoff + typed adapter errors.
3. Native notifications parity on Windows/macOS (Linux already supported).
4. Signed/notarized release artifacts for trust and smoother installs.

## Current status
- Minimal desktop + CLI shipped.
- Multi-provider support shipped.
- Local-first model + keyring-backed secret storage shipped.
- Release automation + checksums + one-command installers shipped.

For internal detailed execution tracking, maintain private repo plans/state.
